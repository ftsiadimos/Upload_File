Upload File

Nginx File Upload App

Modern Flask + Nginx upload app documentation.

View source on GitHub

Features

  • File upload form at /upload
  • Uploads served via /uploads/<filename>
  • Nginx reverse proxy with static upload storage
  • Docker Compose ready for local development

Quick Start

Run the app from the project root with Docker Compose:

docker compose up --build

Then visit http://localhost:9999 in your browser.

Storage

The application stores uploads in ./uploads. Nginx exposes these files from /var/www/uploads so uploaded content is served efficiently.
