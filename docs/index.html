Upload File

A robust file upload system built with Flask, designed for secure and efficient file handling.

Repository Issues Pull Requests

📁 Upload Support

📄

File Upload

Upload individual files with extension validation. Supported types include documents, images, audio, video, archives, and code files.

PDF DOCX XLSX PNG JPG GIF ZIP MP4 MKV MP3 PY JS JSON HTML + more
📂

Folder Upload

Select an entire folder from your local machine. All files and subdirectories are preserved and compressed into a ZIP archive for upload.

✓ Preserves directory structure
✓ Automatic ZIP compression
✓ Multi-file selection
✓ Category assignment

⚡ Features

🔍 Category Management

Organize files into predefined or custom categories. Filter and search across all uploads.

📉 File Management

Download, delete, or move files between categories directly from the file listing.

🔏 Security

Extension validation, secure filename handling, and configurable max file size limits.

📦 Search & Filter

Search files by name and filter by category with an intuitive interface.

🚀 Docker Support

Complete containerized deployment with Docker Compose, Nginx reverse proxy, and Gunicorn.

🎨 Responsive Design

Clean, modern UI that works on desktop and mobile devices.

🛠️ Tech Stack

Backend Flask (Python)
Frontend HTML, CSS
Reverse Proxy Nginx
WSGI Server Gunicorn
Containerization Docker / Docker Compose
Registry GitHub Container Registry (GHCR)

📋 Routes

/ Main file manager UI
/upload File upload POST endpoint
/upload_folder Folder upload POST endpoint
/about About page
/uploads/<path> Serve uploaded files
/download/<path> Download files as attachments

📚 Quick Start

Docker Compose
# Build and start the containers
docker-compose up --build

# Access the application
# http://localhost:9999
Local Development
# Install dependencies
pip install -r requirements.txt

# Run the application
export FLASK_APP=main.py
flask run
