Upload File App

A lightweight Flask app with Nginx-based upload serving, Docker Compose support, and a polished file manager interface.

View source on GitHub

Overview

This repository provides a simple upload manager built with Flask, backed by a shared uploads/ directory, and optionally fronted by Nginx for static file delivery.

Features

  • Upload files from the browser with category support.
  • Download and delete files using safe, sanitized paths.
  • Serve uploaded files through Nginx for performance.
  • Custom categories stored persistently in the upload folder.
  • Graceful error pages for common HTTP failures.

Quick start

Run the application using Docker Compose from the project root.

docker compose up --build

Visit http://localhost:9999 once it is running.

Important routes

  • / — main file manager UI
  • /upload — file upload POST endpoint
  • /uploads/<path> — serve upload files
  • /download/<path> — download files as attachments

Deployment notes

  • Set FLASK_SECRET_KEY in production.
  • Ensure the uploads/ directory is writeable by the container.
  • Use Nginx to serve static upload content instead of proxying it through Flask.
